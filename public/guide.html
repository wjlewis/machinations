Editing State Diagrams

The distinguishing feature of any particular Turing Machine is its state diagram. This section outlines all that you need to know to create and modify your own state diagrams.

All of the controls related to constructing and editing state diagrams appear on the lefthand side of the app, as seen here:

The available controls change depending on the current selection. We will return to this detail later when discussing individual operations.

The most basic state diagram editing operation is adding a new state. To do this, click the "add new state" button (which is always available):

Now, no state will be added after clicking this button. Instead, you should see a "phantom state" underneath your cursor as you move it around the app. To add a state, simply click anywhere on the screen. This adds a new state at the location you clicked:

To select a state, simply click it. The state will change to a purple color to indicate that it is currently selected. To select multiple states, hold the SHIFT key and click each of the desired states. To deselect all selected states, click an empty area on the screen.

States may also be moved around the screen. To move a single state, simply drag it with the cursor. To move a collection of states, select all of them (by holding the SHIFT key) and then drag one of the selected states with the cursor.

To delete a state, select the state and click the "delete state" button:

To delete multiple states, select the desired states and click the appropriate "delete states" button:

In order to simulate a machine's operations, one of the states must be distinguished as the initial state. To do so, select a state and click the "distinguish as initial state" button:
